# 🌿 Plant Leaf Disease Detection using Deep Learning

## 📌 Project Overview

This project detects plant leaf diseases using Deep Learning and Digital Image Processing techniques. It classifies plant leaf images into different disease categories and helps farmers identify diseases at an early stage.

---

## 🚀 Features

- Plant leaf disease detection
- Image preprocessing
- Deep learning prediction
- Disease visualization
- Jupyter Notebook implementation

---

## 🛠 Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook

---

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/udaykumar223/DIP_Project.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook

```bash
jupyter notebook
```

Open

```
Plant Leaf Disease Identification_code.ipynb
```

The model predicts plant diseases from leaf images and displays the prediction.
